# Evolution Timeline Redesign

## Goal
Transform the Evolution Timeline section into an interactive media-rich experience with three main sections, each containing expandable subsections with photos and videos.

## Structure

### Main Sections
1. **Chapter Foundation - Orientation**
   - Overview of chapter establishment
   - Media: Photos and videos from orientation

2. **Chai Links**
   - Subsections:
     - Ch 0 (with media)
     - Ch 1 (with media)
     - More to be added
   
3. **Founders Unplugged**
   - Subsections:
     - Zahid Akhtar (with media)
     - More to be added

## Implementation Plan

### Phase 1: Enhanced Timeline Component
- Create accordion-style sections
- Expandable subsections
- Media gallery within each subsection
- Support for both images and videos

### Phase 2: Data Structure
- JSON-based content structure
- Paths to media files
- Metadata (titles, descriptions, dates)

### Phase 3: Media Display
- Image lightbox viewer
- Video player integration
- Responsive grid layout

### Phase 4: Admin Functionality (Future)
- CMS integration for content management
- Upload interface for media
- Role-based permissions (admin/leads)

## Current Focus
Implementing Phase 1-3 with placeholder content and file paths that admins can update.
